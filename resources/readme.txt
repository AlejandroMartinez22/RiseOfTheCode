En esta carpeta van todos los recursos que permiten reutilizar
ciertos atributos comunes entre distintos nodos. El funcionamiento
es similar al de un objeto. Por ejemplo, el recurso bullet_data define 
los atributos que tiene que tener una bala, luego, cada bala que creamos (archivo .tres)
es una instancia de este recurso. Todas las balas tienen atributos comunes.
